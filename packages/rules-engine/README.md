# Rules Engine

Implements deterministic validation rules for the neutral finance approval demo:

- R-001 duplicate invoice
- R-002 vendor existence
- R-003 vendor bank-account match
- R-004 amount consistency
- R-005 required fields
- R-006 previous payment status
- R-007 supporting evidence requirement

Prior-work note: Finance Reviewer influences this package as a truth-layer pattern only. No prior source code is copied.
